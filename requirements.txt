## ğŸ› Troubleshooting

### CORS Error
If you see CORS errors, make sure:
1. Flask-CORS is installed: `pip install flask-cors`
2. Backend is running on port 5000
3. Frontend correctly references `http://localhost:5000`

### Port Already in Use
```bash
# Kill process on port 5000 (backend)
# On Windows:
netstat -ano | findstr :5000
taskkill /PID <PID> /F

# On Mac/Linux:
lsof -ti:5000 | xargs kill -9
```

### Module Not Found
```bash
# Ensure all packages are installed
pip install -r requirements.txt

# Or install individually
pip install Flask flask-cors pandas numpy scikit-learn
```

## ğŸ¨ Customization

### Change Port
In `app.py`, modify the last line:
```python
app.run(debug=True, host='0.0.0.0', port=5000)  # Change 5000 to your port
```

Update `API_URL` in `index.html`:
```javascript
const API_URL = 'http://localhost:5000';  // Update port here
```

### Modify ML Model
In `app.py`, change the prediction model:
```python
from sklearn.ensemble import RandomForestRegressor

# Replace LinearRegression with RandomForestRegressor
model = RandomForestRegressor(n_estimators=100, random_state=42)
```
